## Installation
1. Clone the project
 ``
    git clone git@github.com:ab-sharifi21/personal-portfolio.git
 ``

2. Create .env file with all the variables from `.env.example`

3. Install the dependencies with ``npm install``

4. Run the application with ``npm run dev``
